Основні команди Git

1. Ініціалізація репозиторію

git init — створює новий Git-репозиторій у поточній папці.

2. Додавання змін до індексації

git add [ім'я_файлу] — додає файл до індексації.

git add . — додає всі файли в поточній папці.

3. Робота з комітами

git commit -m "Опис змін" — створює коміт із повідомленням.

4. Перегляд статусу репозиторію

git status — показує поточний стан робочої директорії.

5. Перемикання між гілками

git checkout [назва_гілки] — перемикає на вказану гілку.

git checkout -b [назва_гілки] — створює і перемикає на нову гілку.
