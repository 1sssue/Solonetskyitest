CSS (Cascading Style Sheets) — це мова стилів, яка використовується для опису вигляду веб-сторінок. Вона дозволяє розділити зміст (HTML) і стильове оформлення, що робить код більш організованим і зручним для підтримки. CSS визначає такі аспекти, як кольори, шрифти, відступи, розміри, а також положення елементів на сторінці. Завдяки CSS, веб-розробники можуть створювати адаптивні і привабливі інтерфейси, забезпечуючи кращу взаємодію з користувачем.
