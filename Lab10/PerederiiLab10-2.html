HTML (HyperText Markup Language) — це стандартна мова розмітки для створення веб-сторінок. Вона описує структуру контенту, використовуючи теги, які вказують на різні елементи на сторінці, такі як заголовки, параграфи, зображення, посилання, таблиці та інші. HTML дозволяє визначити, як виглядатимуть елементи на сторінці, але не відповідає за їхній стиль або поведінку, що здійснюється через CSS та JavaScript. HTML є основою всього веб-контенту та є першою мовою, з якої починається розробка будь-якого веб-сайту.
