Мої гіперпосилання

Головна сторінка

YouTube — один із найпопулярніших відеохостингів в Україні.

DeepL — онлайн-перекладач, відомий високою точністю перекладу завдяки використанню технологій штучного інтелекту.

Reddit — одна з найбільших соціальних платформ для обміну інформацією та дискусій.

Canva — онлайн-інструмент для створення дизайнів, який пропонує користувачам безліч шаблонів для створення презентацій, банерів, логотипів, постів для соцмереж і багатьох інших графічних проєктів.

GitHub — популярна платформа для хостингу коду та спільної розробки програмного забезпечення.

YouTube Music — музична платформа від YouTube, яка пропонує доступ до величезної бібліотеки пісень, альбомів, плейлистів та відеокліпів.
